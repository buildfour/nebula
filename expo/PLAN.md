# Space Shooter Game — GDD & Asset Generation

## What I'll Do

**Step 1: Create a detailed Game Design Document (GDD)**
- A comprehensive markdown file covering game overview, mechanics, enemy types, weapons, power-ups, level design, UI/HUD layout, and art direction
- 3 name options for the game
- Detailed descriptions of all 12 image assets needed

**Step 2: Generate 12 game image assets**
Based on the reference image's vibrant cartoon/comic art style with bold outlines and saturated colors:

1. **Player Ship** — Large stylized fighter spacecraft
2. **Enemy Type 1: Drone** — Small colorful crab-like swarm enemy
3. **Enemy Type 2: Scout** — Medium orb-shaped enemy with tentacles
4. **Enemy Type 3: Heavy** — Large armored gunship enemy
5. **Enemy Type 4: Boss** — Massive mechanical boss ship
6. **Space Nebula Background** — Deep space with purple/blue/teal nebula
7. **Asteroid (Large)** — Rocky space debris
8. **Player Laser Bolt** — Blue energy projectile
9. **Explosion Effect** — Colorful burst/explosion sprite
10. **Shield Power-Up** — Glowing shield pickup icon
11. **Power Boost Power-Up** — Energy/lightning pickup icon
12. **HUD Frame Elements** — Health bar, shield bar, score frame UI elements

**Step 3: Set up the app project** with the GDD file and a simple game shell

All assets will use the "2d game art..." prefix in prompts and match the vibrant, cartoon sci-fi aesthetic from the reference image.
